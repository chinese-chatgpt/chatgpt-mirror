## 什么是 ChatGPT 镜像网站？

**ChatGPT 镜像网站**（ChatGPT Mirror）是指将原始官网的内容复制并放置在另一服务器上的网站服务。国内用户无需翻墙就能用~

- **无需翻墙**：直接在国内使用，无需代理工具。
